# README
## Div 基础组件-Div块
## API

属性 | 说明 | 类型 | 默认值 | 必选
----|-----|------|------|------
nowrap | 文本自动换行 | boolean | false | false
type | 类型 | 'normal'/'panel' | 无 | false
children | 子项 | JSX.Element[] | JSX.Element | 无 | false
className | 样式class | string | 无 | false
style | 样式style | React.CSSProperties | 无 | false
