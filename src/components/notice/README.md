# README
## Notice 基础组件-通告栏
## API

属性 | 说明 | 类型 | 默认值 | 必选
----|-----|------|------|------
mode | 模式 | string | 无 | false
className | 样式类名 | string | '' | false
title | 标题 | string | 'Title' | false
content | 内容 | string | 无 | false
noticeModalShow | 是否可见 | boolean | false | false
icon | 按钮 | string | '' | false
onClick | 点击后回调 | Function | 无 | false
