# README
图片组件。该组件仅作为图片展示组件，可用作头像展示等。

### API

| 属性          | 说明                                            | 必填 | 类型                                                       | 默认值        |
| ------------- | ----------------------------------------------- | ---- | ---------------------------------------------------------- | ------------- |
| *url*        | 图片的链接或base64字符串                     | 是   | Array< string >                                              |               |
| *size* | 图片的尺寸                            | 否   | 'sm'\|'md'\|'lg'                                     | 'md' |
| *borderRadius* | 图片圆角百分值              | 否   | number                            | 0 |
| *showLabel* | 渲染选项的模式                                  | 否   | boolean                                    | false   |
| *label* | 标题文本 | 否   | string |               |
