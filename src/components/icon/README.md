## API

属性 | 说明 | 类型 | 默认值 | 必选
----|-----|------|------|------
type | 类型 | string | 无 | false
color | 颜色 | string | 无 | false
data | 内容 | string | 无 | false
size | 尺寸 | 'xxs'/'xs'/'sm'/'md'/'lg'/'md' | 'md' |false
visible | 是否可见 | boolean | true | false
onClick | 点击时回调 | (): void | 无 | false
