# README

评分组件

### API

| 属性                      | 说明                           | 必填 | 类型                                 | 默认值 |
| ------------------------- | ------------------------------ | ---- | ------------------------------------ | ------ |
| count         | 评分总分 | 是 | number            | 5      |
| allowHalf | 是否允许半分 | 否 | boolean | false  |
| defaultValue | 默认值 | 否 | number | -      |
| disabled | 禁用 | 否 | boolean | -      |
| onChange | 评分变化时间 | 否 | (value: number)=>void |        |
