## API

属性 | 说明 | 类型 | 默认值 | 必选
----|-----|------|------|------
mode | 模式 | 'noData'/'client'/'blankPage'/'error'/'review'/'server'/'noNetwork' | 'noData' | false
message | 显示信息 | string | '暂无文件' | false
style | 样式style | React.CSSProperties | 无 | false
