# README

### API

| 属性                      | 说明                           | 必填 | 类型                                 | 默认值 |
| ------------------------- | ------------------------------ | ---- | ------------------------------------ | ------ |
| *url*                    | 链接的url地址                      | 是   | string                                |      |
| *width*                | iframe的样式宽度属性，默认为100%           | 否   | string                              |    |
| *height*             | iframe的样式高度属性，默认为100%                    | 否   | string                       |        |
