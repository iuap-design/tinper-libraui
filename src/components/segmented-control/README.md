# README
# segmentedControl 菜单分段控制，切换菜单，内容区上下滑动至选中模块吸顶

####使用说明####

   属性名           类型                            默认值          描述
   data	           array<{label, children}>       []             菜单数据
   selectedIndex   number                         0              菜单选中项的索引
   onClick         (el: Object, index: number)                   每个菜单被点击的回调
